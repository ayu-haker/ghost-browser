## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the browser:
```bash
npm start
```

## Usage

- Enter a URL in the address bar and press Enter or click "Go"
- Use navigation buttons to go back/forward or refresh
- Click the home button to return to the start page

## Note

This is a fully functional desktop browser that can load any website!
